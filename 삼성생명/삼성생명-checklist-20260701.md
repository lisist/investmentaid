### 3시나리오 적정가 (3년 기준)

```
python tools/financial_rigor_kr.py three-scenario 결과
낙관(+15%/yr, PER 20배): 390,074원 (+3.5%)
중립( +8%/yr, PER 15배): 242,318원 (-35.7%) ← FAIL
비관( -5%/yr, PER  8배):  87,960원 (-76.7%)
```

**시나리오 가정 근거:**
- 낙관: 삼성전자 특별배당+본업 성장, EPS +15%, PER 20배 (삼성그룹 프리미엄)
- 중립: 삼성생명법 리스크 상존, 본업 꾸준 성장, EPS +8%, PER 15배 (보험사 적정)
- 비관: 삼성생명법 통과 + 삼성전자 주가 하락, EPS -5%, PER 8배 (저성장 보험사)

> **중립 시나리오 -35.7%의 의미**: 현재 PER 29.4배→15배로 압축 시 약 -49%의 밸류에이션 수축이 EPS 성장(+26%)으로 부분 상쇄된 결과. 삼성전자 지분 프리미엄이 사라지면 이 수준이 가능.

### SOTP (Sum-of-the-Parts) 분석 — 삼성생명 핵심 평가 방법론

보험사+투자지주 복합 구조에서 표준 PER·세 시나리오는 왜곡될 수 있다. SOTP가 더 합리적:

| 구성요소 | 가치 추정 | 가정 |
|---------|---------|------|
| 보험 본업 | **23조원** | 본업 순이익 2.3조원 × PER 10배 |
| 삼성전자 지분 (72조원) | **43~50조원** | 규제·유동성 할인 30~40% 적용 |
| **SOTP 합계** | **66~73조원** | |
| **현재 시총** | **75.2조원** | |
| **괴리** | **-2.9% ~ +13.9%** | |

**해석**: 삼성전자 지분 할인율을 30%로 보면 SOTP 73조원 ≈ 현재 시총 75.2조원 (거의 적정). 할인율 40% 적용 시 SOTP 66조원 → 현재가 대비 -12.2%. 삼성생명법 통과(+삼성전자 지분 강제 매각) 시 보험 본업 가치 23조원만 남으면 주당 115,000원.

**클라만의 질문**: "시장이 이 주식을 이 가격에 두는 이유는?" → 삼성전자 연동 상승 프리미엄. 이 프리미엄이 영구적인가? → 삼성생명법 리스크가 구조적으로 상존.

**★★★☆☆** — SOTP 기준 현재가는 적정~소폭 고평가 구간. 단순 PER 기준으로는 과도한 고평가처럼 보이지만, 삼성전자 지분 내재가치 감안 시 다름. 핵심 리스크(삼성생명법)가 해소되어야 안전마진 확보 가능.

## 핵심 리스크 (Top 5)

1. **삼성생명법 리스크 (최고 우선순위)**: 보험업법 개정 시 삼성전자 지분(72조원)을 총자산의 3% 한도(약 10.5조원)로 제한 → 60조원 규모 강제 매각 → 삼성전자 주가 충격 + 삼성생명 지분 프리미엄 소멸. 이 단일 시나리오가 주가를 보험 본업 가치(~115,000원)까지 압축 가능.

2. **삼성전자 주가 동조화**: 기업가치의 61.3%가 삼성전자 지분. 삼성전자 주가 20% 하락 시 삼성생명 지분 가치 14조원 감소 → 주가 직접 충격. 반도체 업황이 삼성생명 투자 결과를 좌우함.

3. **IFRS17 일탈회계 잔존 리스크**: 2023~2024년 유배당보험 계약자 배당 부적절 처리 → 과거 실적이 과대 계상되었을 가능성. 금감원 지도로 정상화됐으나 소급 재무제표 수정·배상 청구 가능성 잔존.

4. **보험 본업 성장 둔화**: 2026Q1 보험서비스손익 YoY -7.7% (예실차 손실 확대). CSM 목표(3.2조) 미달 시 본업 가치 재평가. 저출산·고령화로 장기적 신계약 성장 한계.

5. **금리·환율 복합 리스크**: 저금리 환경에서 운용이익 압박. 달러 강세 시 해외채권 환차손. K-ICS 비율이 금리 변동에 민감 (금리 하락 시 부채 현재가치 증가 → 비율 하락).

> 3. 경영진은 **배당 성향 41%→50% 목표 달성 의지는 신뢰하나, 삼성전자 지배구조 도구 역할이 소수주주 이익과 상충할 수 있어** 일부 불신한다.
