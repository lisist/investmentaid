## 6. 클라만식 하방 분석

### 6-1. 청산가치 추정

**그레이엄 순유동자산가치(NCAV) = 유동자산 - 총부채**
```
python tools/financial_rigor_kr.py calc --expr "1730.6-294.7"  → 1,435.9억원
→ 주당 25,746원
```

**보수적 청산가치(현금 100%, 기타유동자산 60%, 비유동자산 40% 회수율)**
```
→ 순청산가치 1,228.9억원, 주당 22,035원
→ 현재가/청산가치 = 2.07배
```

| 기준 | 주당 가치 | 현재가(45,550원) 대비 |
|---|---|---|
| NCAV(그레이엄식) | 25,746원 | 43.5% 하회 |
| 보수적 청산가치 | 22,035원 | **51.6% 하회** |

현재 주가는 보수적 청산가치의 약 2.07배 — 클라만이 강조하는 "청산가치 이하 매수" 조건에는 크게 못 미친다.

### 6-3. 비인기 자산 여부 — 일시적인가 구조적인가

**구조적 요인**:
- 2025년 실적이 상장 당시 가이던스 대비 대폭 미달 → 펀더멘털 훼손 실재
- **락업 해제 오버행**: 상장일 유통가능물량 35.7%, 6개월 38.1%, 12개월 41.4%, **24개월(2027.11경) 66.4%**, 30개월(2028.5경) 100%
- 모터사이클 매출 비중 87.6%(2024년)로 단일 제품군 편중

**일시적 요인**:
- 신규상장 초기 수급 과열에 따른 반작용
- 소형주 유동성 부족으로 인한 변동성 확대

**종합 판단**: 클라만이 선호하는 "일시적으로 버려진 우량자산"이라기보다는, 이익 추정 재조정과 구조적 오버행이 복합된 자산이다.

### 6-4. 촉매 부재 시 자본 기회비용

배당 정책·자사주 매입 계획 확인 불가(사실상 무배당 추정). 순현금 886.8억원(시총의 34.9%)이 존재하나 이를 주주가치로 실현할 명확한 촉매 부재. 락업 해제가 순차 도래하는 향후 2~3년은 물량 부담이라는 역풍까지 고려해야 한다.
