## 1. 성장 제품 현황

### HBM 포지션

| 제품 | 현황 |
|------|------|
| HBM3E | 엔비디아 GB300에 단독 공급 확인 |
| HBM4 | 엔비디아 Vera Rubin 플랫폼 약 70% 수주. 3사 체제이나 지배력 유지 (SK하이닉스 55%, 삼성 28%, 마이크론 17%) |
| HBM4 판매 상황 | 2026년 전체 HBM 물량 완판 상태 — 공급이 수요를 따라가지 못하는 구조 |
| Q1 2026 시장점유율 | 56.4% |

### HBM4E (차세대)

- **2026년 6월 18일 세계 최초 12-High 샘플 출하 완료**
- 핀당 16Gbps, 전력효율 20%+ 향상, 48GB 용량 구현
- 양산 목표: 2027년

### 기타 성장 제품

| 제품 | 상황 |
|------|------|
| CXL 2세대 메모리 | 2027~2028년 출하 예정, LPDDR6-PIM 탑재 |
| 1c DRAM | 2026년 전체 DRAM 캐파 50%+ 전환 목표 |
| NAND | 321단 전환 가속, 운영마진 48% 흑자 전환 확인 |

## 2. R&D 효율

- 2011~2022년 HBM R&D 860억 원 + 시설 1.5조 원 (시장 침체기에도 지속 — 선견지명의 핵심 증거)
- 2028년까지 $1,030억 투자 선언, 용인 클러스터 600조 원(2050년)
- **MR-MUF 패키징 기술**: 수율·열관리의 핵심 독점 기술, 경쟁사 단기 모방 제한
- TSMC-SoIC 협력: 엔비디아 GPU 아키텍처의 핵심 파트너로 내재화
